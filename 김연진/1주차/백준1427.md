# [백준 1427 소트인사이드](https://www.acmicpc.net/problem/1427)

배열을 정렬하는 것은 쉽다.

수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬하는 프로그램을 만들자.

## 코드

```python
n = list(map(int, str(input())))

n.sort(reverse=True)
print(*n, sep="")
```

## 해결 방식

파이썬 내장 정렬 함수 이용

**과정**

1. 입력받은 수를 쪼개 배열에 담는다.
2. 파이썬 내장 정렬 함수를 이용해 오름차순 정렬
3. 정렬된 배열을 이어 출력
