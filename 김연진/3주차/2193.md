# [백준 2193 이친수](https://www.acmicpc.net/problem/2193)

```python
n = int(input())

dp = [0] * (n+1)

for i in range(1, n+1):
    if i == 1:
        dp[i] = 1
    else: 
        dp[i] = dp[i - 1] + dp[i - 2]

print(dp[-1])
```
